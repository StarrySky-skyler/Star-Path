## 1. 游戏介绍

**星途 Star Path**是我开发的第一款**2D RPG 剧情向 校园 恋爱 治愈 游戏**

讲述了**星痕sky**和**星梦yuki**相爱并帮助**yuki**走出抑郁症的故事
